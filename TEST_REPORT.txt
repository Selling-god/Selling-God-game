판매의 신 v12 RELEASE
=======================

가장 쉬운 적용 순서
1. 이 폴더의 모든 파일과 assets 폴더를 GitHub 저장소 루트에 업로드합니다.
2. Supabase SQL Editor에서 supabase_SQL_v12.sql 전체를 한 번 실행합니다.
3. 기존 Supabase 프로젝트가 다르면 config.js의 URL과 ANON KEY만 바꿉니다.
4. Vercel/Render에서 정적 사이트로 다시 배포합니다.

필수 파일
- index.html
- style.css
- game.js
- config.js
- assets 폴더 전체
- supabase_SQL_v12.sql (DB에 한 번 실행)

주요 기능
- 이메일 로그인/회원가입과 계정 저장
- 시작 자금 500,000원
- 500종 거래 아이템
- 전당포 즉시판매/흥정
- NPC 경매 및 유저 입찰
- 유저 중고장터와 NPC 거래 제안
- 주식, 휴대폰, 소장품, 가챠, 하우징
- 반응형 UI와 네트워크/초기화 오류 화면

주의
- assets 폴더를 누락하면 NPC와 대표 아이템 이미지가 표시되지 않습니다.
- Supabase 무료 플랜 사용량을 초과하면 로그인이나 저장이 일시적으로 실패할 수 있습니다.
- 상용 배포 전에는 본인 Supabase 프로젝트에서 회원가입, 구매, 판매, 경매를 실제 계정 2개로 최종 점검하세요.
